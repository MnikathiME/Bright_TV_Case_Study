Overview

This project analyzes BrightTV user and viewership data to uncover insights that support the company’s goal of growing its subscription base.

As a Data Science Student , I used SQL, Databricks, and Excel to transform raw transactional data into meaningful insights and strategic recommendations for the CEO and Customer Value Management (CVM) team.

Business Objective

The main goal of this analysis was to:

Understand user behavior and consumption trends
Identify factors influencing viewership
Recommend content strategies for low-consumption periods
Propose initiatives to grow BrightTV’s user base

Tools & Technologies
SQL
Databricks
Microsoft Excel (Pivot Tables & Dashboards)
Canva
Miro
CSV / Excel datasets
Data Processing & Methodology
Converted raw data from Excel to CSV
Loaded and transformed data in Databricks
Cleaned inconsistent values (e.g. unit price formatting, null values)
Converted timestamps from UTC to South African time
Created calculated fields:
total_amount
TimeOfDay
AgeGroup
Exported processed data to Microsoft Excel for dashboard creation

Key Insights
Midday hours showed the highest viewership, while night-time had lower engagement
Sports content (e.g. major events) drove peak consumption but lacked consistency
Adults dominate the user base, with low participation from youth segments
Gauteng, Western Cape, and KwaZulu-Natal contribute the majority of viewership
Certain days and time slots present growth opportunities for engagement

Recommendations
Content Strategy
Introduce local entertainment content during low-viewership periods
Expand sports-related programming beyond live events
Develop family and youth-focused content

Growth Strategy
Launch student and family subscription packages
Target underserved provinces with localized campaigns
Offer flexible subscription plans (mobile, weekend, pay-per-view)
Invest in exclusive and original South African content
Implement referral and loyalty programs


Closing Summary

This project demonstrates how data analytics can transform raw viewership data into actionable insights. The analysis supports data-driven decisions that can improve engagement, optimize content strategy, and drive sustainable growth for BrightTV.
